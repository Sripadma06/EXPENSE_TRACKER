# Expense Tracker Application

## Overview

Expense Tracker is a full-stack personal finance management application designed to help users monitor their income, expenses, and overall financial activities. The system provides an organized way to track transactions, categorize spending, and gain insights into financial habits.

## Features

### User Management

* User Registration and Login
* Secure Authentication
* Profile Management

### Expense Management

* Add Income Records
* Add Expense Records
* Update Transactions
* Delete Transactions
* View Transaction History

### Financial Tracking

* Expense Categorization
* Monthly Expense Summary
* Income vs Expense Analysis
* Budget Monitoring

### Dashboard

* Financial Overview
* Recent Transactions
* Spending Statistics
* Category-wise Expense Tracking

## Database Setup

1. Create a MySQL database.
2. Configure database credentials in application.properties.
3. Run the Spring Boot application.
4. Access the application through the browser.
